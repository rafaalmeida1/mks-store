### Ferramentas utilizadas

- Nextjs
- Redux Toolkit
- Radix-ui
- Styled Components
- Jest 
- Testing Library
- Axios


### Para Configurar o Front-end

```bash
npm install
# or
yarn
```

#### Para iniciar o front-end de fato:
```bash
npm run server
# or
yarn server
```

### Executando os testes
```bash
npm run test
# or
yarn test
```
